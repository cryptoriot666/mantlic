# Mantlic: Building an AI Agent for DeFi in 72 Hours

## The Problem
DeFi is powerful but confusing. Multiple UIs, gas optimization, slippage tolerance... what if you could just type "swap 100 MNT for USDC"?

## What We Built
Mantlic: an autonomous AI trading agent for Mantle DeFi. Natural language → on-chain execution. No clicks.

## Architecture
- Next.js 16 + React 19 + Tailwind CSS v4
- ERC-8004 agent identity on Mantle Sepolia
- DeepSeek AI for natural language understanding
- Three.js + GSAP for award-worthy animations
- 1inch Fusion for optimal swap routing

## Key Technical Decisions

### Why ERC-8004?
We wanted verifiable agent identity. Not just a username — an on-chain NFT. Every benchmark score, every decision, every attestation verifiable by anyone.

### Why terminal UI?
Speed. Traders don't want to click through 5 screens. One command, one result.

### Why Mantle?
Low fees + fast finality = DeFi that feels like CeFi.

## The Hackathon Experience
Built in 72 hours. Learned that terminal-only UX is actually a feature for Track 4 (Consumer & Viral DApps) because it creates shareable moments.
