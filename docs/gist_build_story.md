# Mantlic: The 72-Hour DeFi Agent Build Story

**Mantlic** — an autonomous AI trading agent for Mantle DeFi. Type a command, get a result. No clicks.

## The Problem
DeFi is powerful but confusing. Multiple UIs, gas optimization, slippage tolerance... what if you could just type "swap 100 MNT for USDC"?

## What We Built
- Natural language → on-chain execution
- ERC-8004 agent identity (on-chain NFT)
- Terminal UI for speed
- Built in 72 hours for Mantle Hackathon

## Tech Stack
- Next.js 16 + React 19 + Tailwind CSS v4
- DeepSeek AI for NLP
- Three.js + GSAP for animations
- 1inch Fusion for swap routing
- Mantle Sepolia testnet
